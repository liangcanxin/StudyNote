*补：最常用 DOS命令*

| DOS命令     | 作用                           |
| :---------- | :----------------------------- |
| cd 目录路径 | 进入一个目录                   |
| cd ..       | 进入父目录                     |
| dir         | 查看本目录下的文件和子目录列表 |
| cls         | 清除屏幕命令                   |
| 上下键      | 查找敲过的命令                 |
| Tab键       | 自动补齐命令                   |
