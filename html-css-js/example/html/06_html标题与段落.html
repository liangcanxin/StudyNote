06、html标题与段落

标题

标题是双标签

写法<h1></h1>...<h6></h6>

在一个网页中，h1标题最重要，并且一个html文件中建议出现一次h1标签，虽然使用多次在浏览器中也是可以解析的，但是这样做不规范

h5、h6标签在网页中不经常使用

段落

段落也是双标签

写法<p></p>

示例

        <h3>html解释</h3>
        <p>HTML 指超文本标签语言。HTML 是通向 WEB 技术世界的钥匙。</p>

展示效果

html解释

HTML 指超文本标签语言。 HTML 是通向 WEB 技术世界的钥匙。
