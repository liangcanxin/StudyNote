14.div和span

div标签

div标签是块元素，多用做一个区域块的划分，一般配合css实现分割区域

span标签

span标签是内联元素，一般不做区域划分用，而是用作文本的修饰

        <div style="height: 32px; background-color: #555555;">这是块1</div>
        <div style="height: 64px; background-color: #777777;">这是块2</div>
        <div><span style="font-weight: bold; background-color: #999999;">这是块3</span>，但是被修饰着</div>
