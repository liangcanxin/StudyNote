08.图片标签

写法<img src="引入图片的地址" alt="提示信息" title="提示信息" width=npx height=mpx />

  • src:引入图片的地址url
  • alt:当图片出现问题无法显示时，可以显示一段友好的提示信息
  • title:当将光标移入图片时，显示与图片相关的提示信息
  • width、height:图片的大小，当添加上图片的大小时，网速慢无法立即显示出图片的情况下，会自动为图片留足空间，这样做用户体验会更好一些（相对于没有写width和height）

路径的引入

  • 相对路径（一般情况下建议使用）
  • 绝对路径
