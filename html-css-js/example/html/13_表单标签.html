13.表单标签

基本标签

form、input、textarea、select、label ...

input标签

该标签为单标签，由一个type属性，决定该标签为什么控件

其他属性

表单标签常见的一些其他属性有checked、disabled、name、for ...

示例

        <form action="https://www.baidu.com" method="POST">
            <!-- action属性定义在提交表单时执行的动作。向服务器提交表单的通常做法是使用提交按钮，通常，表单会被提交到web服务器上的网页。如果省略action属性，则action会被设置为当前页面 -->
            <!-- method属性规定在提交表单时所用的http方法(get或post) -->
            <fieldset>
                <legend>Personal information：</legend>
                <!-- fieldset元素组合表单中的相关数据，legend元素为fieldset元素定义标题 -->
                username：
                <!-- 以下为文本输入表单 -->
                <input type="text" name="username" placeholder="username">
                password:
                <input type="password" name="password" placeholder="password">
                <!-- 如果要正确地被提交，每个输入字段必须设置一个name属性 --> < br >
                sex:
                <!-- 以下为单选按钮输入表单 -->
                <input type="radio" name="sex" value="male" checked>male    
                <input type="radio" name="sex" value="female">female<br>
                own vehicle:
                <input type="checkbox" name="vehicle" value="bike">bike    
                <input type="checkbox" name="vehicle" value="car">car
                <input type="checkbox" name="vehicle" value="disabled" disabled>disabled chocie<br>
                <select name="sports" > 
                    <option value="choices" selected disabled>--please choose your favorite sport--</option>
                    <option value="basketball" selected > basketball < /option > 
                    < option value="football" > football < /option > 
                    < option value="tennis" > tennis < /option > 
                </select > < br >
                <textarea name="information" cols="30" rows="10" > the information about ... < /textarea > < br >
                <input type="submit" value="submit">
            </fieldset >
        </form >
        <button type="button" onclick="alert('Hello World!')"> click me! < /button >
