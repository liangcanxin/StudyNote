07.html文本修饰

强调

强调是双标签，有两种方式

写法1：<strong></strong>

写法2：<em></em>

区别

  1. 写法和展示效果是有区别的，写法1的展示效果是文本加粗，写法2的展示效果是文本倾斜
  2. strong的强调性更强，em的强调性稍弱

示例

        <strong>这是加粗强调</strong>
        <em>这是倾斜强调</em>

显示效果

这是加粗强调

这是倾斜强调

上标与下标

上标写法：<sub></sub>

下标写法：<sup></sup>

示例

X<sup>1</sup>=Y<sub>2</sup>

显示效果

X1=Y2

删除文本与插入文本

删除文本写法：<del></del>

插入文本写法：<ins></ins>

注：一般情况下，删除文本都是和插入文本配合使用的

示例

        <p>原价：<del>￥99.00</del>    现价：<ins>￥59.00</ins></p>

显示效果

原价：￥99.00    现价：￥59.00
