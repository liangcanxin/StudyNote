11.列表标签

无序列表

示例

        <p>中国</p>
        <ul>
            <li>广东省</li>
            <li>广西省</li>
            <li>福建省</li>
            <li>江苏省</li>
        </ul>

展示效果

中国

  • 广东省
  • 广西省
  • 福建省
  • 江苏省

快捷键【li*5+tab/回车键】- >快速生成5个li双标签，同时无序列表符合嵌套规范，既可以其中再嵌套列表等

有序列表

与无序列表相似，唯一不同的是使用的是ol标签和li标签

示例

        <p>中国</p>
        <ol>
            <li>广东省
                <ul>
                    <li>广州市</li>
                    <li>深圳市</li>
                    <li>东莞市</li>
                </ul>
            </li>
            <li>广西省</li>
            <li>福建省</li>
            <li>江苏省</li>
            <li>...</li>
        </ol>

效果展示

中国

  1. 广东省
    • 广州市
    • 深圳市
    • 东莞市
  2. 广西省
  3. 福建省
  4. 江苏省
  5. ...

定义列表

与另外两种列表相似，不同的是使用的是dl、dt和dd标签

示例

        <dl>
            <dt>html</dt>
            <dd>html决定网页的内容</dd>
            <dt>css</dt>
            <dd>css决定网页的样式</dd>
        </dl>

展示效果

html
html决定网页的内容
css
css决定网页的样式

注：列表之间可以相互嵌套，形成多层级的列表
