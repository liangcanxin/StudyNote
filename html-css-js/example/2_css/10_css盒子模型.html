10. css盒子模型

css盒子模型组成

  • content(可以理解为物品)
  • padding(可以理解为填充物)
  • border（可以理解为包装盒)
  • margin(可以理解为盒子与盒子之间的间距)

具体介绍

content

表示内容区域，由width和height组成其大小

padding

内边距，也称内填充，具体写法：

  • 只写一个值：30px(表示上下左右)
  • 写两个值：30px 40px(表示上下、左右)
  • 写四个值：30px 40px 50px 60px(表示上右下左)

单一样式，如果设置其大小单位，只能写一个单位值

  • padding-left
  • padding-right
  • padding-top
  • padding-bottom

margin

外边距，也称外填充，具体写法：

  • 只写一个值：30px(表示上下左右)
  • 写两个值：30px 40px(表示上下、左右)
  • 写四个值：30px 40px 50px 60px(表示上右下左)

单一样式，如果设置其大小单位，只能写一个单位值

  • margin-left
  • margin-right
  • margin-top
  • margin-bottom

注：padding和margin的区别

  1. 背景色填充到margin以内的区域（不包括margin区域）
  2. 文字在content区域添加
  3. padding不能为负数，而margin可以为负数

box-sizing

盒尺寸，可以改变盒子模型的展示形态

默认值

content-box：width、height -> content

border-box：width、height -> content、padding、border

更改默认的使用场景

  • 不用再去计算一些px值
  • 解决一些100%分配出界的问题

盒子模型的一些问题

  • margin叠加问题，出现在上下margin同时存在的时候，会取上下中值较大的作为叠加的值
  • margin传递问题，出现在嵌套的结构中，只是针对margin-top的问题

扩展知识

  • margin左右自适应是可以的，但是上下自适应是不行的（需要额外操作）
  • 在盒子嵌套中，对于内部盒子，width、height不设置，对盒子模型的影响，会自动去计算容器的大小，节省代码
