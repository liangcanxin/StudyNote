07. css选择器

(1) id选择器

写法：#elem{} ->id="elem"

快捷键【div#elem+tab】 ->【<div id="elem">】

注意

  1. id是唯一的，一个id在一个页面中只能出现一次，出现多次是不符合规范的
  2. 命名的规范，由字母、下划线、中划线和字母组成，其中，命名的第一个字符不能是数字
  3. 命名写法
    • 驼峰写法：seachSmallButton/SearchSmallButton
    • 短线写法：search-small-button
    • 下划线写法：search_small_button

(2) class选择器

写法：.elem{} ->class="elem"

快捷键【div.elem+tab】 ->【<div class="elem"】

注意

  1. class选择器是可以复用的
  2. 可以添加多个class样式(<div class="box content">)
  3. 多个样式的时候，样式的优先级根据css编写的顺序决定，而不是class属性中的顺序
  4. 标签+类的写法

(3) 标签选择器(tag选择器)

写法：div{} -> <div></div>

使用场景

  • 去掉某些标签的默认样式时使用
  • 复杂的选择器中，如层次选择器

(4) 群组选择器(分组选择器)

可以通过英文逗号的方式，给多个不同的选择器添加统一css样式，来达到代码的复用

(5)通配选择器

写法：*{} -> div，ul，li，p，h1，h2...

注：尽量避免使用通配选择器，因为会给所有的标签添加样式，慎用

使用的场景：去掉所有标签的默认样式时

(6)层次选择器

  • 后代->    m n{}
  • 父子->    m>n{}
  • 兄弟->    m~n{}当前m下面的所有兄弟n标签
  • 相邻->    m+n{}当前m下面的相邻的n标签

(7)属性选择器

写法：M[attr]{}

注：匹配属性

  • =：完全匹配
  • *=：部分匹配
  • ^=：其实匹配
  • $=：结束匹配
  • [][][]：组合匹配

伪类选择器

m:伪类{}

  • :link    访问前的样式    (只能添加给a标签)
  • :visited    访问后的样式    (只能添加给a标签)
  • :hover    鼠标移入时的样式    (可以添加给所有标签)
  • :active    鼠标按下时的样式    (可以添加给所有标签)
  • :after/:before    通过伪类的方式给元素添加一段文本内容，使用content属性
  • :checked/:disabled/:focus    都是针对表单元素的
  • 结构性伪类选择器
    • :nth-of-type()/:nth-child()    角标是从1开始的，1表示第一项，2表示第二项|n值表示从0到无穷大，可用于表格的隔行换色，如2n/2n+1等
    • :first-of-type/:last-of-type
    • :only-of-type
    • :nth-of-type()/:nth-child()    type：类型；child：孩子

注：

  1. 如果以上前四个伪类要都可以生效，一定要注意顺序：LVHA
  2. 对于前四个伪类，一般网站只这样去设置： a{}    a:hover{}
