01. css基础语法

写法

选择器{属性1:值1; 属性2:值2; ... 属性n:值m;

常用属性

  • width:选择器的宽度
  • height:选择器的高度
  • background-color:选择器的背景色

网页常用的长度单位

  1. px:像素
  2. %:百分比

解释

外容器1：600px    当前容器50%：300px

外容器2：400px    当前容器50%：200px

css样式引入方式

内联样式

内联样式是写在html标签内的，通过style属性调用；例如：<p style="height: auto;"></p>

内部样式

  • 内部样式一般写在head标签中，采用style标签包围起来。例如：<style>div{height: auto;}</style>
  • 优势：内部样式可以实现html标签的样式复用、复合W3C的规范标准，同时实现网页的html结构和css样式分离

外部样式

  • 外部样式单独建立一个css文件
  • 引入方式1：通过link标签引入外部样式，rel属性指定资源跟页面的关系，href属性指定资源的地址，一般将link标签放置于head标签中；例如：<link rel="stylesheet" href="../reset.css">
  • 引入方式2：通过@import方式引入外部样式，但是这种引入方式存在不少问题，不建议使用，具体原因：参见

css中的颜色表示法

单词表示法

单词：red、blue、green、yellow...

十六进制表示法：0123456789abcdef，如#4e4f44

rgb三原色表示法

例如rgb(255,255,255)，取值范围0~255

颜色提取方式：使用fhhelper插件或者Photoshop颜色提取器
