心痛！贺州一医生因过度劳累猝死，年仅32岁

2月28日，广西贺州市八步区灵峰镇卫生院副院长钟进杏因日夜奋战在抗疫一线，劳累过度猝死在卫生院宿舍，因公殉职，年仅32岁。

2月29日，灵峰镇卫生院院长李雨训告诉记者，灵峰镇是八步区靠近广东最近的小镇，近期全院医务人员都参与到企业复工复产疫情防控工作中。

2月27日，钟进杏和同事们研究讨论下一步工作计划，一起忙到晚上11时多，计划次日出差到八步区南乡镇。

2月28日上午9时多，钟进杏仍未现身，同事打他电话也没接。院长李雨训立即赶到他的宿舍，在门外叫喊，但无人回应；拨打他的电话，宿舍里传来电话铃声。

李雨训立即破门而入，发现钟进杏躺在床上一动不动，经检查已无生命体征，初步诊断系劳累过度猝死在宿舍。

“他还这么年轻，我们至今接受不了。”李雨训说，钟进杏去年从八步区铺门镇卫生院调到灵峰镇卫生院任副院长，到岗后就重新开展住院部业务，挑起基层医院业务工作的大梁，平时工作兢兢业业。

疫情发生后，钟进杏更是主动承担危险系数最大的发热门诊和疑似病例住所消毒杀毒等工作，工作上积极主动、任劳任怨的他，得到单位同事和广大群众的一致认可。

灵峰镇作为八步区向东开放的东大门，同时也是两广交界交通最繁忙的主干道接入点。为筑牢防控墙，守护好身后八步区75万老百姓的健康安全，灵峰镇在疫情防控期间设有两个省际体温检测检查站，防控任务很重。

从1月26日起，钟进杏主动放弃轮休，一直驻守在防控工作一线，与同事坚守发热门诊、下村入户排查、跟踪随访湖北返乡人员检测体温、宣传、消毒、调配物资、为坚守体温监测点医院一线医护人员送关爱小餐、深入企业指导复工复产等工作。事发前一天，钟进杏已经奋战在防控一线33天。

据了解，钟进杏已成家，有一名6岁的女儿。

2月29日，当地有关领导前往钟进杏家中慰问他的家人。目前，灵峰镇卫生院已经在为钟进杏做工伤认定申请材料。
