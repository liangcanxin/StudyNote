13. 标签嵌套规范

常规嵌套

  • ul li
  • ol li
  • dl dt dd
  • table tr td

块能够嵌套内联

        <div>
            <span></span>
            <a href="#"></a>
        </div>

块嵌套块（不一定能）

        <div>
            <div></div>
        </div>

以下嵌套即不可以，是错误写法

        <p>
            <div></div>
        </p>

内联是不能嵌套块（大多数不能）

以下是错误示范

        <span>
            <div></div>
        </span>

特例，以下写法正确

        <a href="#">但是a标签不能嵌套a标签
            <div></div>
        </a>
