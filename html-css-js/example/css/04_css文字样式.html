04. css文本样式

字体类型

写法：选择器{font-family:字体类型;}

分类1：英文or中文

分类2：衬线体、非衬线体

注意点

  • 多个字体类型的设置目的
  • 字体类型中引号的添加目的

字体大小

写法：选择器{font-size:number(px)|单词(small...不推荐使用)

默认字体大小为16px

字体加粗

写法：选择器{font-weight:值;}

默认值为normal，可选值为bold或者数字

注意：当值为数字时，100~500的字体效果跟normal一样，但是600~900字体开始加粗

字体样式

写法：选择器{font-style:值}

常用选值有normal、italic

注：oblique也是表示斜体，但是用的比较少，一般了解即可

两种斜体类型的区别

  • italic带有倾斜属性的字体才可以设置倾斜操作
  • oblique是没有倾斜属性的字体也可以设置倾斜操作
