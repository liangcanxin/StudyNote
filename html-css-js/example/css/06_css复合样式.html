06. css复合样式

写法

复合样式的写法是通过空格的方式实现的。复合写法有的是不需要关系顺序的，例如background、border；而有的是需要关系顺序的，例如font

  1. background:red url() repeat 0 0;
  2. border:1px red solid
  3. font: 注：最少要有两个值  size family(这两个值是有先后顺序的)
    • font:weight style size family
    • font:style weight size family
    • font:weight style size/line-weight family

注：如果非要复合样式和单一样式混合去写的话，那么要先写复合样式，再写单一样式，这样单一样式才不会被覆盖掉
