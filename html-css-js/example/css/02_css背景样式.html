02.css背景样式

背景色

写法：选择器{background-color:颜色值;}

背景图

写法：选择器{background-image:url(图片地址);}

默认：会水平垂直都铺满背景图

背景图平铺方式

写法：选择器{background-repeat:值;}

取值可以是repeat-x、repeat-y、repeat（默认值）、no-repeat

背景位置

写法：选择器{background-position:背景位置;}

背景位置可以输入px值或者%或者单词

  x轴方向上取值left、center、right

  y轴方向上取值top、center、bottom

背景图滑动or固定

写法：选择器{background-attachment:值;}

取值可以是scroll（默认值，背景位置是按照当前元素进行偏移的）或者fixed（背景位置是按照浏览器进行偏移的）
