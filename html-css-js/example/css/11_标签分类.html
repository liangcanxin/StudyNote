11. 标签分类

按类型：

  • block:div p ul li h1 ...
    1. 独占一行
    2. 支持CSS所有样式
    3. 不写width的时候，跟父元素的width相同
    4. 所占区域是一个矩形
  • inline:span a em strong img ...
    1. 挨在一起
    2. 有些样式不支持，例如：width、height、margin、padding
    3. 不写width的时候，宽度由内容决定
    4. 所占的区域不一定是矩形
    5. 内联标签之间会有空隙，原因：换行产生的，可以通过写在同一行（不过一般不用inline布局，也就是可以不用考虑它。如果一定要处理，可以在其父元素加上font-size：0，然后在该元素中加入font-size：16px）
  • inline-block:input select ... （挨在一起，会有空隙，但是支持宽高）

注：布局一般用块标签，修饰文本一般用内联标签

按内容

  • FLOW：六内容/li>
  • Metadata：元数据
  • Heading：标题
  • Sectioning：分区
  • Phrasing：措辞
  • Embedded：嵌入的
  • Interactive：互动的

按显示

  • 替换元素：浏览器根据元素的标签和属性，来决定元素的具体显示内容 img、input...
  • 非替换元素：将内容直接告诉浏览器，将其显示出来 div、h1、p...
